Factorielle recursive

Énoncé

Implémente une fonction récursive factorielle(n) en C qui calcule n! (n factorielle).

  • 0! = 1
  • 1! = 1
  • n! = n × (n-1)!
Exemple de sortie attendue
sortie.txt 
factorielle(5)  // 120
factorielle(0)  // 1
factorielle(7)  // 5040
Code de départ
factorielle.c 
#include <stdio.h>

int factorielle(int n) {
    // ton code ici
    return 0;
}

int main() {
    printf("%d\n", factorielle(5));
    printf("%d\n", factorielle(0));
    return 0;
}
factorielle_correction.c 
#include <stdio.h>

int factorielle(int n) {
    if (n <= 1)
        return 1;
    return n * factorielle(n - 1);
}

int main() {
    printf("%d\n", factorielle(5));  // 120
    printf("%d\n", factorielle(0));  // 1
    printf("%d\n", factorielle(7));  // 5040
    return 0;
}
