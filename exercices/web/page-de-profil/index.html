Page de profil

Énoncé

Crée une page HTML complète représentant un profil fictif. La page doit utiliser des balises sémantiques :

  • <header> pour le nom et la photo de profil.
  • <main> pour la bio et la liste des compétences.
  • <footer> pour les liens de contact.

Pas de CSS requis pour cet exercice — concentre-toi uniquement sur la structure HTML sémantique.

Exemple de sortie attendue
apercu.html 
<!-- Structure attendue -->
<header>
  <h1>Alice Dupont</h1>
  <img src="photo.jpg" alt="Alice Dupont" />
</header>
<main>
  <p>Développeuse passionnée...</p>
</main>
<footer>
  <a href="mailto:alice@example.com">Email</a>
</footer>
Code de départ
profil.html 
<!DOCTYPE html>
<html lang="fr">
<head>
  <meta charset="UTF-8">
  <title>Mon Profil</title>
</head>
<body>
  <!-- ton code ici -->
</body>
</html>
profil_correction.html 
<!DOCTYPE html>
<html lang="fr">
<head>
  <meta charset="UTF-8">
  <title>Profil — Alice Dupont</title>
</head>
<body>

  <header>
    <h1>Alice Dupont</h1>
    <img src="photo.jpg" alt="Photo de profil d'Alice Dupont" width="120">
    <p>Développeuse web — Paris, France</p>
  </header>

  <main>
    <section>
      <h2>À propos</h2>
      <p>Passionnée par le développement web et l'open source.</p>
    </section>
    <section>
      <h2>Compétences</h2>
      <ul>
        <li>HTML & CSS</li>
        <li>JavaScript</li>
        <li>Python</li>
      </ul>
    </section>
  </main>

  <footer>
    <h2>Contact</h2>
    <a href="mailto:alice@example.com">alice@example.com</a>
    <a href="https://github.com/alice">GitHub</a>
  </footer>

</body>
</html>
